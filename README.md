## Project setup
### backend
```
$ cd backend
```
```
$ pip install virtualenv
```
```
$ virtualenv venv
```
```
$ source venv/bin/activate 
```
```
$ pip install -r requirements.txt
```
### frontend
```
$ cd frontend
```
```
$ npm install
```
```
$ npm run build
```
## Project start
```
$ python3 manage.py 
```
